# SwiftUIAndReduxExample
[ING] - SwiftUIとReduxで作るサンプルアプリ

SwiftUIとReduxを組み合わせたUI実装サンプルになります。
基本的には、APIからのデータ取得＆表示やLocalのDBへのデータ登録処理等をする形となっています。

※ Backend側はKotlin + Spring Bootで動かしています。
